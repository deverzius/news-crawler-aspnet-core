## Installation
1. Clone the repository
   ```sh
   git clone https://github.com/deverzius/news-crawler-aspnet-core
   ```

2. Change directory and build the project
   ```sh
   cd news-crawler-aspnet-core
   dotnet build
   ```

3. Run the project
   ```sh
   dotnet watch
   ```
